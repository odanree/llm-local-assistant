## 🎯 Problem Statement

**The Interaction Deadlock (from v2.11.0 logs)**:

```
[Executor] Running step 1/2: npm install
[Executor] Iteration 1/3: Attempt executing...
[Executor] npm ERR! Proceed? (y/n)    ← Prompt detected
[Executor] Retrying in 5 seconds...
[Executor] Iteration 2/3: Attempt executing...  ← Infinite loop
[Executor] Retrying in 5 seconds...
[Timeout] 30 seconds elapsed, killing process
```

**Root Cause**: `spawnSync()` is blocking. No async event loop to receive user input. Executor retries automatically because it doesn't recognize prompts.

---

## 🏛️ Solution Architecture

### Current (v2.11.0): Synchronous Blocking
```
┌─ executeStep(step)
│  └─ spawnSync(cmd)           ← Blocks everything
│     └─ Waits for process exit
│        └─ No event loop to handle prompts
│           └─ Retries (TRAP!)
└─ Next step
```

### New (v2.13.0): Asynchronous Event-Driven
```
┌─ executeStep(step)
│  └─ ProcessHandle.spawn(cmd)  ← Non-blocking
│     ├─ onData: Check for prompts
│     ├─ onPrompt: SUSPEND
│     └─ onExit: Resume or complete
│        └─ Event loop free for user input
└─ Next step (async/await)
```

## 📋 Implementation Checklist

### Phase 1: Refactor executeStep() - RUN Case (Day 1)

**Location**: `src/executor.ts`, line ~2280

**Changes**:
- [ ] Replace `const result = cp.spawnSync(...)` with `const handle = await this.commandRunner.spawn(...)`
- [ ] Add ProcessHandle event handlers (onData, onPrompt, onExit)
- [ ] Implement Circuit Breaker pattern for prompt detection
- [ ] Remove retry logic from this block (it's now in the async loop)
- [ ] Return Promise<ExecutionResult> instead of synchronous result

**Expected Code**:
```typescript
// OLD (Blocking):
const result = cp.spawnSync(command, args, { shell: true });
if (result.status !== 0) {
  // Retry logic (TRAP)
}

// NEW (Async):
const handle = await this.commandRunner.spawn(command, {
  cwd: this.projectRoot,
  timeout: this.config.timeout,
});

return new Promise((resolve, reject) => {
  handle.onData((chunk) => {
    if (this.streamingIO.detectPrompt(chunk)) {
      this.plan!.status = PlanState.SUSPENDED_FOR_PERMISSION;
    }
  });

  handle.onExit((code) => {
    resolve({ status: code === 0 ? 'success' : 'failed' });
  });
});
```
